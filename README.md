# Messenger Real Time Chat App

A Beginners swift project, a realtime chat app in Swift 5 using Firebase

## Features

- Facebook Login
- Google Sign in
- Email/password registration/login
- Realtime conversations
- Photo messages
- Video Messages
- Location Messages
- Search users
- Deleting conversations
- User profile
- Darkmode support


## Dependencies


- Firebase/Core
- Firebase/Auth
- Firebase/Database
- Firebase/Storage
- Firebase/Crashlytics
- Firebase/Analytics
- FBSDKLoginKit
- GoogleSignIn
- MessageKit
- JGProgressHUD
- SDWebImage
